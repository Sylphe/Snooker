  - total units / completion size

- Logging fields:
  - average units completed per attempt
  - best attempt
  - completion count
  - optional highest break
- KPIs:
  - average completion %
  - best attempt
  - completions
  - highest break

UX / data quality:
- Delete confirmation wrapper
